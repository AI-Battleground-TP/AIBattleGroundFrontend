# Bu proje bir Node.js (Vite + React + TypeScript) uygulamasıdır.
# Python paketi yoktur; `pip install -r requirements.txt` kullanılmaz.
#
# Gerçek kurulum listesi:
#   - package.json  → doğrudan bağımlılıklar (sürüm aralıkları)
#   - package-lock.json → kilitlenmiş (çözülmüş) tüm paket ağacı
#
# Önerilen kurulum (tekrarlanabilir):
#   cd blind-test-frontend
#   npm ci
#
# Geliştirme için hızlı kurulum:
#   npm install
#
